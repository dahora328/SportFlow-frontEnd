# React + TypeScript + Vite

TO SportFlow FrontEnd fornece telas para cadastrar, listar, editar e remover
atletas, integrando-se a uma API REST configurável. O projeto prioriza uma DX
rápida com Vite (HMR), tipagem estática com TypeScript e estilização por
utilitários com Tailwind CSS. O objetivo é entregar um fluxo simples e eficiente
para gerenciamento de atletas no navegador.

Tecnologias usadas React (biblioteca de UI).

TypeScript (tipagem estática).

Vite (dev server e build).

Tailwind CSS (estilização utilitária).

ESLint (linting de código).
